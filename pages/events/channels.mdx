# Channels

A channel points to one or more topics in Radical. A topic can contain one or more event types. Channels are common for both outbound and inbound events.

To create a channel, follow these steps:

1. Go to the Events tab.
2. Click on "Add" and select the Channels category. Enter a name and confirm by clicking the plus button.
3. Specify the topic name where messages have to be consumed or produced.
4. Associate the channel with only one [binder](./binders).
5. Describe whether the channel has one type of event or multiple types.
