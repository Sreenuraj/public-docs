# Outbound Event

What is an outbound event?

If a microservice wants to send/publish an event to another service, an outbound event can be used to send data.

Steps to create an outbound event:

1. Go to the Events tab.
2. Click on "Add" and select the Outbound Event category. Enter a name and confirm by clicking the plus button.
3. Choose the channel (for more explanation, see [Channels](./channels)).
4. Choose the schema of the outbound event.
5. All fields are selected by default, and you can choose to modify this.
